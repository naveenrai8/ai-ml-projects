# ai-ml-projects

## Documentation

### UV
#### Initialize UV

```bash
# initialize the project
uv init .
```

#### Add MCP to project dependencies

```bash
uv add "mcp[cli]"
```

#### Install MCP 

Use Claude desktop here. Need to restart it everytime existing functionality is being updated or new being added.
```bash
mcp install main.py
```
